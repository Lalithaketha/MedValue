Anaemia

  • Deficiency : vitamin B12,B9
  • vitamin Name : Folate
  • Source : Ferrous sulfate
  • Food : Red meat, poultry, Seafood, Beans, Dark green leafy vegetables, Peas.
  • Body part Affected : Heart

Night Blindness

  • Deficiency : vitamin A
  • vitamin Name : Retinol
  • Food : red and green (leafy) vegetables, carrots, sweet potatoes, mango, papaya
  • Body Part Affected : Eyes

Pellagra

  • Deficiency : Vitamin B3
  • Vitamin Name : Niacin
  • Food : meat, poultry, fish, nuts, and legumes.
  • Body Part Affected : Brain and Nervous System

Beriberi

  • Deficiency: Vitamin B1
  • Vitamin Name : Thiamine
  • Food : peas, some fresh fruits, nuts, wholegrain breads, liver.
  • Body Part Affected : cardiovascular system

Scurvy

  • Deficiency: Vitamin C
  • Vitamin Name : Calciferol
  • Source : Citrus Fruits
  • Food : Oranges and orange juice, peppers, strawberries, potatoes.
  • Body Part Affected : bleeding gums, loosened teeth and bleeding under your skin.

Rickets

  • Deficiency: Vitamin D
  • Vitamin Name : Calciferol
  • Source : Sunlight
  • Food : Fresh fatty fish, liver, some mushrooms and egg yolks
  • Body Part Affected : bone development in children

Goitre

  • Deficiency: Iodine
  • Source : Salt
  • Food : Strawberries, Cranberries and Pineapples
  • Body Part Affected : thyroid gland
