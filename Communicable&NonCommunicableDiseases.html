ChickenFox

  • Type : Communicable
  • Cause : Virus
  • Treatment : Calamine lotion and a cool bath with added baking soda, uncooked oatmeal, or colloidal oatmeal
  • Anti Viral Drug : Acyclovir(Zovirax)
  • Body part Affected : Entire Body

COVID-19

  • Type : Communicable
  • Cause : SARS-CoV-2
  • Treatment : Adequate food and proper medicine related to particular symptoms
  • Anti Viral Drug : Remdesivir
  • Body part Affected : Respiratory tract

Measles

  • Type : Communicable
  • Cause : Virus
  • Treatment : Fever reducers, Antibiotics, Vitamin A
  • Body part Affected : First on the face and behind the ears, then spreads downward to the chest and back and finally to the feet

HIV

  • Type : Communicable
  • Cause : Virus
  • Treatment : There is no cure for HIV, but you can control it with HIV treatment.
  • Body part Affected : CD4 helper cell or T cell

Cancer

  • Type : Non-Communicable
  • Cause : Changes (mutations) to the DNA within cells
  • Treatment :Surgery, Chemotherapy, Radiation Therapy, Targeted Therapy, Immunotherapy.
  • Body part Affected : Bones, liver, or lungs

Diabetes

  • Type : Non-Communicable
  • Cause : Sugar builds up in the bloodstream.
  • Treatment : Metformin (Fortamet, Glumetza, others)
  • Body part Affected : Eyes, heart, nerves, feet and kidneys

HyperTension

  • Type : Non_Communicable
  • Cause : Unhealthy lifestyle choices, such as not getting enough regular physical activity
  • treatment : Calcium Channel Blockers, Angiotensin Converting Enzyme inhibitors, Angiotensin Receptor Blockers, Diuretics.
  • Body part Affected : Heart, brain, kidneys, and eyes
